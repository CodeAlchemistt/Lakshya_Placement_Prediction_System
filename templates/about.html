System Architecture

The Lakshya Placement Prediction System is an end-to-end machine learning application designed to forecast the likelihood of university students securing campus placements.

The Machine Learning Model

At the core of this system is a Random Forest Classifier, an ensemble learning method constructed via Scikit-Learn. The model evaluates over 20 distinct student metrics—ranging from CGPA and coding scores to hackathon participation and mock interview performance.

The Tech Stack

  • Backend API: Python 3, Flask
  • Data Science: Pandas, NumPy, Scikit-Learn
  • Frontend UI: HTML5, Tailwind CSS, Jinja Templating
  • Deployment: Gunicorn, Hosted on Railway Cloud
